HTML

  • The head element contains information about the webpage
  • The body element shows the content that is visible and the user can interact with.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • you can add colors in using either their name like blue or gray, or use their hex #
  • when you want to address everything you can use a * symbol to have everything fall under a certain rule

Git

  • git status checks what branch we are currently working in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git switch (which branch here) sends you to a specific branch

JavaScript

  • Using === is a strict equality operator. It checks to see if two values are equal.
  • Const is used to make something constant
  • the keyword Let allows us to reassign the value of a variable over time
  • Array is used to hold a group of data, normally they hold data that is related
  • Index are numbers assigned to data items
